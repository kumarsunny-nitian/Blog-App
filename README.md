# Blog App

**GitHub Repository:** https://github.com/kumarsunny-nitian/Blog-App.git

A basic full-stack blog application where users can create, edit, view, and delete blog posts using CRUD operations and RESTful APIs.

---

## Features

- Create blog posts
- Edit posts
- Delete posts
- View all blogs
- REST API based operations

---

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- HTML
- CSS
- JavaScript

---

## Languages Used

- EJS
- JavaScript

---

## Installation

Clone the repository:

```bash
git clone https://github.com/kumarsunny-nitian/Blog-App.git
cd Blog-App
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
node index.js
```
